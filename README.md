# （SHAPE SHIFT）

## ゲームの内容
（〇・△・□の形状の特性を生かしてゴールに向かうゲームです。
　〇は移動、△は攻撃、□は防御と各形状の役割を明確にすることで、単なるアクションゲームではなく、各特性を生かした頭脳戦が必要になるものになっています。）

（<img width="1187" height="587" alt="image" src="https://github.com/user-attachments/assets/4ef17435-3fcf-43c5-968a-79ea79170133" />）

## 操作説明
- スペースキーで形状変化
- 〇状態:矢印キーで移動
- △状態:矢印キーに入力した方向に攻撃
- □状態:シフトキーを押して防御
